1. Header tag: This tag is used to define the header or the initial section of a web page. It usually includes the site title, logo, navigation menu, and other important page information.
2. Nav tag: This tag is used to define the navigation menu or links related to the web page. This tag helps users easily navigate through the site and access different pages.
3. Article tag: This tag is used to define an article or the main content of a web page. It contains essential and useful content for users and generally includes a title, images, text, and related links
4. Section tag: This tag is used to group different sections of a web page. It is styleable with CSS and helps to display the web page in an organized and structured manner.
5. Footer tag: This tag is used to define the footer or the final section of a web page. It includes contact information, related links, copyright notice, and other final page information.
